## Contributing to Documentation

Found a typo? Missing information? We welcome improvements!

### Quick Fixes
- Edit the page directly and submit a PR
- Use the "Edit on GitHub" link (if available)

### Larger Changes
1. Read [Documentation Guide](contributing/documentation-guide.md)
2. Check open issues with `documentation` label
3. Follow the [Documentation Templates](../DOCUMENTATION_PLAN.md#documentation-templates)

### Documentation Standards
- Use clear, concise language
- Include runnable code examples
- Add links to related topics
- Test all examples before submitting
