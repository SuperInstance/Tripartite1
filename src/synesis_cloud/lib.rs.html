1// synesis-cloud: Cloud connectivity and QUIC tunnel for SuperInstance AI platform
2//
3// This crate provides:
4// - QUIC tunnel with mTLS for secure cloud communication
5// - Cloud escalation client for offloading queries to cloud LLMs
6// - Billing client with local-first usage tracking
7// - LoRA upload and hot-swap functionality
8// - Collaborator system for sharing projects
9// - Telemetry and heartbeat system

11//! # Cloud Mesh Infrastructure
12//!
13//! This crate provides the complete cloud connectivity layer for the SuperInstance AI platform.
14//! It enables secure QUIC tunnels, billing management, LoRA sharing, and collaboration features.
15//!
16//! ## Main Components
17//!
18//! - **Tunnel**: QUIC-based bidirectional communication with state machine
19//! - **Escalation**: Client for offloading queries to cloud LLMs
20//! - **Billing**: Cost tracking with tier-based pricing (Free/Managed/BYOK)
21//! - **LoRA**: Upload, hot-swap, and cloud management of LoRA adapters
22//! - **Collaborator**: Invite-based collaboration system
23//! - **Telemetry**: Device vitals collection and heartbeat protocol
24//! - **Protocol**: Binary message protocol for tunnel communication
25//! - **Streaming**: Server-sent events for real-time responses
26//!
27//! ## Architecture
28//!
29//! The cloud mesh follows a local-first approach:
30//! 1. All billing is tracked locally before cloud sync
31//! 2. LoRAs can be managed locally and uploaded to cloud
32//! 3. Collaborator invites are generated locally
33//! 4. Escalation contexts are built locally before transmission
34//!
35//! ## Thread Safety
36//!
37//! All shared state uses `Arc<RwLock<T>>` for async-safe mutations.
38//! Metrics use `Arc<AtomicU64>` for lock-free counting.
39//!
40//! ## Usage
41//!
42//! ```rust,no_run
43//! use synesis_cloud::billing::client::BillingClient;
44//! use synesis_cloud::billing::types::BillingTier;
45//!
46//! # fn main() -> Result<(), Box<dyn std::error::Error>> {
47//! // Create billing client
48//! let billing = BillingClient::new(
49//!     "api-key".to_string(),
50//!     BillingTier::Free { monthly_limit_cents: 1000 }
51//! );
52//!
53//! // Calculate cost
54//! let cost = billing.calculate_cost("claude-sonnet", 1000, 500)?;
55//! # Ok(())
56//! # }
57//! ```
